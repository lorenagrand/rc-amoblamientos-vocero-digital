# RC Amoblamientos — Vocero Digital con IA (HeyGen)

Pieza de video-marketing generada íntegramente con IA para **RC Amoblamientos**,
fábrica de muebles y amoblamientos a medida (Cerrito, Entre Ríos).
Entrega del módulo de producción multimedia — Creación de Contenido con IA (Coderhouse).

## 🎬 Video final
▶️ https://drive.google.com/file/d/1rJqvkiBQ-Wvs0uw1ibAiJ7YqmMB_2N5h/view?usp=sharing

## 🛠️ Workflow de producción

| Activo | Herramienta | Detalle |
|--------|-------------|---------|
| Guion | Claude | Guion persuasivo ~30s, con pausas para parpadeo |
| Voz | ElevenLabs (Multilingual v2) | Voz de stock en español, tono cálido |
| Personaje | Gemini (Módulo 3) | Valeria, personaje consistente de marca |
| Avatar / Lip-sync | HeyGen (plan gratis, 720p) | Photo Avatar + audio importado |
| Música | Udio | "Madera que abraza tu hogar", nivelada al ~15-20% |
| Edición final | CapCut | Integración de banda sonora |

## 📝 Guion utilizado

> En RC Amoblamientos no fabricamos muebles. Diseñamos el lugar donde tu vida
> sucede. Hace más de 30 años que transformamos madera en hogares: cocinas a
> medida, vestidores, placares… cada pieza pensada para vos. Esto no es un
> catálogo. Es tu casa, antes de existir. RC Amoblamientos. El arte de habitar.

## 🎙️ Descripción de la voz
Voz femenina en español, tono cálido y aspiracional, velocidad ligeramente
reducida para un registro premium y pausado. Audio limpio (sin ruido de fondo)
para garantizar un lip-sync natural.

## 🔄 Proceso de sincronización
Se importó el MP3 de ElevenLabs directamente a HeyGen (no se usó la voz nativa)
para controlar el tono. HeyGen mapeó fonemas→visemas sobre el Photo Avatar.
Las pausas del guion permiten micro-movimientos faciales naturales (parpadeo).

## 🎨 Nota de consistencia de marca
El personaje (Valeria), la paleta monocromática cálida y el tono de voz
respetan el briefing de marca y el Manual de Identidad Visual (Módulo 3).
Nota de producción: se usó un encuadre 3/4 coherente con el estilo
editorial/lifestyle de marca; una v2 a cámara mantendría el mismo bloque de
consistencia (vestuario, iluminación, paleta).
